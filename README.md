# Hey, I'm Akshat 👋

EEE undergrad at AKTU who somehow ended up writing more code than circuit diagrams. Somewhere along the way backend development became the thing I actually care about — designing REST APIs, managing databases, writing clean service layers, and making sure everything actually works when deployed. It's the behind-the-scenes work that most people never see, and I find that genuinely interesting.

Currently working on two things. **Forensick** is a 2D psychological horror survival game built in Unity — the kind of game where darkness slowly closes in around you, your torch dims over time, and hallucinations start appearing the longer you survive. It started as a 10-day holiday project and turned into something I'm actually proud of. The other is a **Task Manager API** built with Spring Boot, complete with JWT authentication, layered architecture, global exception handling, MySQL integration, and deployed live on Render. Both projects are on my GitHub if you want to poke around.

On the backend side I work with Java, Spring Boot, Spring Security, and JWT for auth. Database work happens in MySQL with Spring Data JPA — schema design, relationships, query optimization basics. For deployment I use Render and have worked with Docker on a basic level. Game development is Unity 2D with C#, which turns out to share more with backend thinking than you'd expect. My usual tools are Postman for API testing, Git for version control, IntelliJ for Java, and VS Code for everything else.

New tech genuinely excites me in a way that's probably unhealthy — I'm the kind of person who discovers something interesting at 10pm and is still reading about it at 2am. Currently grinding DSA in Java alongside everything else, because apparently I don't believe in having free time. Always looking for ways to build things that are actually useful or at least genuinely interesting.

Open to internships, collaborations, and problems worth solving.
